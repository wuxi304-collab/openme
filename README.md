# OpenMe

> 打开文件，不必先猜该用哪个软件。

OpenMe 是一个本地优先的桌面文件工作台：把常见文档、表格、图片、压缩包、代码与工程文件放进同一个安静、统一的界面中预览和检查。它以 Electron、React 和 TypeScript 构建，采用深色工作台与轻量游戏化视觉，但文件处理始终是主角。

## 为什么做 OpenMe

日常工作里的文件常常来自不同软件：一份 PDF、一张图、一张表、一个 ZIP，偶尔还有 DWG。OpenMe 希望减少应用切换，让“先看懂文件，再决定怎么处理”变成一个顺畅的动作。

- **本地优先**：预览流程尽可能留在设备上，避免把文件默认上传到未知服务。
- **一个工作台**：最近文件、标签页、搜索、缩放和快捷键保持一致。
- **安全预览**：对 HTML、SVG、Office 转换内容和压缩包采取隔离与路径保护。
- **诚实兼容**：明确区分稳定预览与实验性支持，不把近似渲染当作原生软件效果。

## 已支持的内容

| 类别 | 格式与能力 |
| --- | --- |
| 文档 | PDF、Markdown、DOCX、纯文本与代码 |
| 数据 | CSV、JSON、XLSX，支持搜索、排序或分页 |
| 图片 | 常见位图与 SVG，支持缩放、平移、旋转、适应窗口和 1:1 |
| 压缩包 | ZIP 文件列表、文本预览与安全解压 |
| 媒体 | 音频、视频与字体预览 |
| 电子书 | EPUB 安全文本阅读 |
| 工程文件 | STEP / IGES 等实验性 3D 预览；DWG 实验性语义预览 |

### 关于 DWG

DWG 是封闭且版本复杂的格式。OpenMe 当前使用开源解析链路提供辅助预览，适合快速检查，不保证字体、标注、代理对象、布局和复杂实体与 AutoCAD 完全一致。对于生产签审和精确编辑，请仍以原生 CAD 软件为准。

## 快速开始

需要 Node.js 20 或更新版本。

```powershell
npm install
npm run electron:dev
```

构建 Windows 版本：

```powershell
npm run dist
```

仓库也保留了面向普通用户的中文启动与部署脚本。

## 常用快捷键

- `Ctrl + O`：打开文件
- `Ctrl + K`：打开命令面板
- `Ctrl + W`：关闭当前标签
- `Ctrl + Tab`：切换标签

## 技术栈

- Electron + React + TypeScript + Vite
- PDF.js、Mammoth、JSZip、read-excel-file
- LibreDWG Web / ACadSharp 辅助链路
- OCCT Import JS + Three.js

## 路线

- 继续提升稳定格式的交互一致性、无障碍和大文件表现
- 为格式能力建立可复现的样例与回归测试
- 探索更可靠、可自由分发的 DWG 渲染方案
- 在明确授权、可预览与可撤销的前提下引入 LLM 辅助修改流程
